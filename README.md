# Poker Points

> A FigJam plugin for agile story point estimation — fast, visual, and right where your stories live.

![Icon](icon.svg)

---

## About

Poker Estimation lets your team assign story point values directly to stickies in FigJam. No separate tool, no context switching — estimates appear as colored badges on your story stickies, grouped so they move together.

### Estimation schemes included

| Scheme | Values |
|---|---|
| **Fibonacci** | 1, 2, 3, 5, 8, 13, 21, ? |
| **Modified Fibonacci** | 0, ½, 1, 2, 3, 5, 8, 13, 20, 40, 100, ? |
| **T-Shirt Sizes** | XS, S, M, L, XL, XXL |

All schemes are **fully editable** — add, remove, or change values and they persist across sessions.

---

## How to use

1. Open the plugin from the FigJam plugin menu
2. Choose your estimation scheme using the tabs at the top
3. Assign a value to a sticky in one of two ways:
   - **Drag** a value card directly onto a sticky — a colored badge snaps to it
   - **Select** one or more stickies, then click a value to stamp them all at once
4. Re-assign at any time — the badge updates automatically
5. Select multiple estimated stickies to see the **total and average** in the summary bar
6. Use the **✏ Edit** button to customise the values in any scheme

---

## Setup

### Load in FigJam
1. Open FigJam
2. Go to **Main Menu → Plugins → Development → Import plugin from manifest**
3. Select the `manifest.json` file from this folder
4. Run the plugin from the Plugins menu
